The Space Situational Awareness Ontology

View on GitHub Download .zip Download .tar.gz

Description

The SSA ontology (SSAO) files. The overall project is to apply ontology to the orbital domain in order to foster data-sharing, integration among orbital debris, space object and SSA catalogues and databases; automated reasoning, knowledge discovery and semantic and systems interoperability. As such it is related to the other repositories, each of which is conceived as either an ontology module in an ontology architecture for the overall orbital domain, or as a sub-domain part of a single ontology encompassing the overall domain.

Status

Under development, depending on opportunities to sustainably work on it. Subject to revision, merger with other ontologies in the Orbital Space Domain Ontology Project
